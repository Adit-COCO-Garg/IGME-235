CyberPunk 2077
The game takes inspiration from Cyberpunk 2077. The game references the theme of the world with human enhancements. The game also references CyberPunk color themes, and in general the CyberPunk genre.
Kimetsu No Yaiba
The game takes inspiration from the humulity and respect present in Kimetsu No Yaiba as Tanjiro feels sorry and humility towards the deamons. And, this emotion invokes a form of release and freedom in the deamons he kills. The game will feature a similar set of emotions.
